Final Project Proposal!

Self Watering Planter by Mai Li Jorgensen

Project Concept/Motivation:

I want to make a system that will detect whether a plant in a potter is wet or dry. It will then automatically spray some water on it if it is too dry.

Anticipated Bill of Materials:

  • Flower and pot
  • Water level detection sensor
  • Arduino and wires
  • Servo motor and string
    • Create a pulley to trigger the spray bottle
  • Tall spray bottle (or a stand to get the bottle to the pot level)
  • Resistors
  • Box for the system to be housed in
  • LEDs/speaker (contingency)

Timeline:

  • Week 1: create the circuit with servo motor/string/spray bottle and arduino/sensor
  • Week 2: build the housing unit, find plant/potter to use, spray bottle
