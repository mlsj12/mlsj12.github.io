Resistance Calculations:

5v - 1.8v = 20mA × R
3.2v = 20mA × R
R = 3.2v / 20mA
R = 3.2v / 0.02m
R = 160Ω

In the final circuit design, I selected a 220Ω resistor. According to the calculation, the circuit requires a minimum resistance of 160Ω to ensure a current of 20mA. Using a resistor with a slightly higher value than the minimum provides more protection of keeping the current lower than 20mA while the circuit still performs effectively.

Arduino Code:


// HCDE 439
// Mai Li Jorgensen
// A1: Blink!

void setup() {
 // set up code: on / off for different times
 // pins 1 - 3 are treated as an output in this program
 pinMode(1, OUTPUT);
 pinMode(2, OUTPUT);
 pinMode(3, OUTPUT);
}

void loop() {
 // main code that repeats forever: on/off for different durations
 // pin 1 will blink at 100ms and then turn off until the loop completes and starts over
 // LED connected to pin 1 is turned
 digitalWrite(1, HIGH);
 // LED stays on as program pauses for 100ms
 delay(100);
 // LED in pin 1 is turned off
 digitalWrite(1, LOW);
 // program delays for 100ms
 delay(100);

 // pin 2 will blink at 500ms and turn off until the loop starts over
 // LED connected to pin 2 is turned on
 digitalWrite(2, HIGH);
 // program delays for 500ms
 delay(500);
 // LED in pin 2 is turned off
 digitalWrite(2, LOW);
 // program delays the same amount (500ms)
 delay(500);

 // pin 3 will blink at 1000ms (1 second) and turn off until the loop starts again
 // LED in pin 3 will turn on
 digitalWrite(3, HIGH);
 // program stops for a second
 delay(1000);
 // LED 3 is turned off
 digitalWrite(3, LOW);
 // program stops for another second
 delay(1000);
}

2: How many LEDs could you blink independently with your Arduino? How much current would that draw?

For the first question, this would depend on how many individual pins there are on the Arduino. To do more than pin available would be possible by adding more LEDs to the circuits, but that wouldn't be an independently run LED anymore. Looking at the controller, I could blink 14 different LEDs in the 0-13 numbered pins that are programmable. As for the current, each LED is its own circuit that should have no more than 20mA going through it. So if I were to use all 14 pins for 14 LEDs, all using 20mA or less, I would have about 20mA x 14 circuits = 280mA. Running all 14 pins, I should not exceed 280mA of current through the whole Arduino to keep it in a safe range.
