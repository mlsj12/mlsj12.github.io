JetSetPump:

Concept

For my final project, I designed a self-watering plant system that ensures plants receive the right amount of water over an extended period, especially when the owner is away, such as on vacation or a business trip.

This system will use a water pump, water sensor, and an Arduino to monitor and automate watering. When the soil moisture drops below a certain threshold, the system will activate the pump to deliver water from a reservoir directly to the plant.

The product combines technology and sustainability to give users a better way to take care of their plants by only needing to fill up the water reservoir when it it low.

The power coming from the wall is controlled by the MOSFET transistor that is being powered by the Ardunio like the sensor. With little power, it can turn on and off the pump's power of 12v.

Technical Implementation

For my JetSetPump system, I have designed an automated self-watering plant system that monitors and maintains soil moisture levels.

The input device in my system is a water sensor, which continuously measures the amount of moisture that is in the soil. When the sensor detects that the water level has dropped below a predefined threshold, it sends a signal to the Ardunio, triggering the output device, a water pump that is connected to a reservoir and the potted plant.

My code ensures the pump waters the plant until the soil moisture reaches enough water mositure for the plant. From previous testing of the pump, I determined the pump moves between 275 (completely submerged in water) and 300-ish (as dry as possible). In this 25 step difference, I preset the pump activation at 80% dryness (292). This makes sure the plant receives adequate hydration without the risk of overwatering.

Code Snippet

 
// variable for the sensor and pump
int sensorpin = 7;
int mostfet = 6;

// variable for the sensor output
int value = 0;

void setup() {
  // start serial monitor
  Serial.begin(9600);
  // initialize the sensor pin
  pinMode(sensorpin, OUTPUT);
  pinMode(mostfet, OUTPUT);
}

void loop() {
  // start the sensor
  digitalWrite(sensorpin, HIGH);

  // write water level to serial
  value = analogRead(A0);
  Serial.print("Water Value: ");
  Serial.println(value);

  // stop the sensor
  digitalWrite(sensorpin, LOW);
  
 
  // fully submerged = ~275 | as dry as can be = ~ 300
  // if plant is 70% dry, give it some water
  if (value > 292) {
    // rotate pump for a second
    digitalWrite(mostfet, HIGH);
    delay(1000);
    // turn off pump
    digitalWrite(mostfet, LOW);
  }

  // check plant water level again
  if (value > 292) {
    // rotate pump for a second
    digitalWrite(mostfet, HIGH);
    delay(1000);
    // turn off pump
    digitalWrite(mostfet, LOW);
  } else {
    // if water is good, delay 2 seconds before loop repeats
    delay(2000);
  }
}

Physical Implementation

After I got the input and outputs sorted and the system was working each time, I started on the physical aspect of the system. While not pretty, I made a stand and compartments for the Ardunio and "inners" to hide all that clutters the system visually.

I started with the pump, it's very heavy on one side as it's a motor essentially. Despite the weight, I wanted it to be directly above the cleaned out All Purpose Cleaner bottle I used. So I started with stabilizing the bottle to hold the extra weight on the top.
I soon realized I also needed support for the pump itself and added a platform coming out of the bottle's neck and supports downward as the cardboard wouldn't hold the entire pump. While the motor

was supported here, I decided to make a stand for the motor that is rounded, so it sits nicely on top and is easy to put back after refilling the water reservoir.

Lastly, I put a cover over the stand and put the Ardunio and other wires inside behind the bottle to tidy everything up.
